# 🎓 Student Management System (SMS) — Django Project

A full-featured **University Student Management System** built with Django.  
This project manages students, courses, exams, results, and role-based access for Admin, Teachers, and Students.

---

## 🚀 Features

### 🔐 Authentication & Roles
- Custom user model (Admin / Teacher / Student)
- Secure login & logout system
- Role-based access control (RBAC)
- Protected dashboards

### 👨‍🏫 Admin Panel
- Manage students, teachers, courses
- View system overview dashboard
- Full control over academic structure

### 🎓 Student Module
- View profile and academic details
- View results and transcripts
- Search system integration

### 🧑‍🏫 Teacher Module
- Manage marks entry
- Assign grades
- View assigned courses

### 📊 Academic System
- Course management
- Exam system
- Marks & GPA calculation (weighted)
- Result generation system

### 📄 Transcript System
- HTML transcript view
- PDF transcript generation (ReportLab)
- Search-based student transcript access

---

## 🛠️ Tech Stack

- **Backend:** Django (Python)
- **Database:** SQLite (Development)
- **Frontend:** HTML, CSS (Django Templates)
- **PDF Engine:** ReportLab
- **Authentication:** Django Custom User Model
